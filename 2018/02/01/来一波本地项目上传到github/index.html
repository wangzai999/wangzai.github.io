来一波本地项目上传到github

1.找到要上传的项目文件夹，并打开命令行

2.git初始化文件夹

3.将代码上传到缓存区

4.将代码推送到本地库

5.在github上新建项目，创建好后把地址拷贝

6.将本地仓库关联到github上

7.将代码进行合并（这步是为了防止下一步上传代码报错）

8.上传代码到github（如果是第一次上传代码，会弹出输入账号密码的窗口）

已经上传好了，当修改文件时，只需要 : 上传缓存区->上传本地库->上传到GitHub
